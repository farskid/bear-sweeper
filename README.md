# Bear Sweeper

Kid-friendly Minesweeper — no numbers.

**Play:** https://farskid.github.io/bear-sweeper/

Sleeping bears hide under the grass. Open safe squares, follow paw prints, leave honey so bears stay asleep. Wake too many and you lose your berries.

## Themes

| | Kids 🐻 | Crystal 💎 |
|---|---|---|
| Danger | Sleeping bears | Dark gems |
| Hint | Color + paw direction | Color + crystal facets |
| Marker | 🍯 honey | ✨ sparkle |
| Lives | 🫐 berries | 💠 crystals |

## Maps

🌱 Meadow → 🌿 Woods → 🌲 Forest → ⛰️ Mountain → 🌙 Night

## Play locally

Open `index.html` in a browser — no build step.

## Stack

Single-file vanilla HTML / CSS / JS.
